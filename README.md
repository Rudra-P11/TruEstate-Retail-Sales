# Retail Sales Management System

A comprehensive **React + Express + MongoDB** application for managing and analyzing retail sales data. The system provides an intuitive dashboard for viewing, filtering, sorting, and searching through large-scale sales records with advanced filtering capabilities.

![Dashboard](<Screenshot 2025-12-08 040238.png>)

**Project Statistics:**
- **Total Records:** 785,000 sales transactions
- **Database:** MongoDB Atlas
- **Frontend:** React 18 with TypeScript & Tailwind CSS
- **Backend:** Express.js with TypeScript
- **Architecture:** Client-Server with RESTful API
